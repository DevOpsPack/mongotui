# Installation

## Requirements

- Python 3.11+
- MongoDB 4.4+ (standalone, replica set, or Atlas)

## Install from PyPI

```bash
pip install mongotui
```

## Install from source

```bash
git clone https://github.com/DevOpsPack/mongotui.git
cd mongotui
pip install -e .
```

## Verify installation

```bash
mongotui --help
```
